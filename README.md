# Курс по компьютерному зрению ВШЭ и МФТИ

Добро пожаловать в репозиторий курса по компьютерному зрению для студентов Высшей школы экономики и Московского физико-технического института.

## Правила курса

### Формирование итоговой оценки

Оценка за курс складывается из двух компонентов:
- Домашние задания (ДЗ) — 70%
- Экзамен (Э) — 30%

Финальная оценка вычисляется по формуле:
```
ОЦЕНКА = 0.7 * ДЗ + 0.3 * Э
```
с последующим математическим округлением до целого числа.

### Оценка домашних заданий

Общая оценка за домашние задания рассчитывается по формуле:
```
ДЗ = max(10, S / (N - K))
```

Где:
- **N** — количество домашних заданий, которое будет выдано в курсе (предположительно по одному на каждое занятие)
- **S** — сумма всех баллов, полученных за выполненные домашние задания
- **K** — количество домашних заданий, которые могут быть прощены (не учитываются при расчете)

Важные особенности:
- За каждое домашнее задание можно получить больше максимального количества баллов (по усмотрению проверяющего)
- Предполагается, что максимальный балл за одно задание равен 10, но может незначительно варьироваться

### Сроки сдачи и штрафы

- Для каждого домашнего задания установлен стандартный дедлайн — **2 недели** с момента выдачи
- Дедлайн может быть изменен или перенесен — об этом объявляется отдельно
- При отсутствии специальных объявлений действует стандартный срок в 2 недели

После дедлайна домашние задания принимаются с понижающим коэффициентом:
- **0.8** — первая неделя просрочки
- **0.6** — вторая неделя просрочки
- **0.4** — третья и последующие недели просрочки

### Блокирующие элементы

И домашние задания, и экзамен являются **блокирующими** для получения финальной оценки, за исключением случаев получения автомата.

### Автоматы

Если накопленная оценка за домашние задания достаточно высока, студент может получить автомат:
```
Накоп = Округление(0.7 * ДЗ)
```

Условия получения автомата:
- Если **Накоп ≥ 6**, студент может получить эту оценку как итоговую **без сдачи экзамена**
- Студент **не может** получить оценку "Отлично" без сдачи экзамена


### Использование GPT, а так же инструментов для вайбкодинга

Мы понимаем, что в текущих реалиях странно и глупо запрещать студентам пользоваться лучшими достижениям технологического развития последних лет, поэтому решили не ограничивать вас в использовании подобного инструментария. Но мы просим вас в комментариях кратко указывать, что из инструментов вы использовали. **Основным** же требованием является **понимание** кода, который вы сдаете в качестве решения. Если на экзамене вы не сможете детально прокомментировать, что сделано в сданном домашнем задании, то это будет считаться нарушением правил, ведущим к понижению оценки и за часть домашнего задания, которую вы не можете объяснить, и за экзамен.
